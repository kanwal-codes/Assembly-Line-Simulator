# Assembly Line Simulator - Frontend

React + TypeScript frontend for the Assembly Line Simulator dashboard.

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

The frontend will be available at `http://localhost:3000`

## Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- **Dashboard**: Real-time statistics and charts
- **Orders**: View all orders (completed/incomplete)
- **Stations**: Monitor station activity and inventory
- **Statistics**: Detailed analytics and visualizations

## Tech Stack

- React 18
- TypeScript
- Vite
- Recharts (for visualizations)
- React Router
- Axios (HTTP client)
